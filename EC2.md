## EC2 Fundamentals

- EC2 = Elastic Cloud Compute
- EC2 is a virtual machine within the cloud and is referred to as an instance within AWS.

## Types of Instances
- General Purpose - Balance of compute, memory, and networking. 
- Compute Optimized - Great for compute intensive task. (HPC/ML/GAMING)
- Memory Optimized - Fast performance for workloads that will process large datasets in memory.
- Storage Optimized - Ideal for storage intensive tasks that require high sequential reads/writes to large data sets on local storage.

## Purchasing Options
- On-Demand - Pay-as-you-go, consumption based model. 
- Reserved (1 or 3 yr commitment) Ideal for long workloads. 
- Savings Plan (1 or 3 yr commitment) Commmitment to a certain amount of usage.
- Spot Instances (90% cheaper than on-demand) Ideal for non-critical short workloads that can be interrupted. 
- Dedicated Host - The ability to book an entire physical server.
- Dedicated Instances - No other customer shares your underlying hardware.
- Capacity Reservations - Reserve capacity in a specific AZ for any duration.
